需求接收 → 需求准备 → 需求挖掘 → 需求分析 → 需求验证 → 需求沟通 → 需求管理
```

## 阶段一：需求接收（5分钟）

### 📥 步骤1.1：初步接收需求
**场景：** 用户/产品经理找到你说："我们需要加个功能..."

**行动清单：**
- [ ] 不要立即说"好"或"不行"
- [ ] 快速记录用户的原始表述
- [ ] 记录提出人、时间、紧急程度

### 🤔 常见问题1：需求描述太模糊怎么办？

**问题表现：**
- "我要一个AI功能"
- "系统太慢了，优化一下"
- "加个类似竞品的功能"

**AI解决路径：**
```
第一步：AI快速扩展问题
└─ Prompt: "这个需求很模糊：'[原始需求]'，请帮我生成10个澄清问题，
   涵盖背景、目标、场景、约束、预期效果"

第二步：AI生成的问题示例（直接问用户）
├─ 背景类：为什么现在提出这个需求？遇到了什么问题？
├─ 目标类：期望达到什么效果？如何衡量成功？
├─ 场景类：谁在什么情况下使用？频率如何？
├─ 约束类：时间要求？预算限制？技术栈限制？
└─ 预期类：有参考案例吗？最低可接受标准是什么？

第三步：拿着问题与用户沟通
└─ 记录用户的回答
```

**工具推荐：** ChatGPT、Claude、文心一言

---

### 🤔 常见问题2：用户直接提解决方案怎么办？

**问题表现：**
- "我需要一个搜索框"
- "给我加个导出Excel的按钮"
- "用GPT-4来实现"

**AI解决路径：**
```
第一步：识别这是解决方案而非需求
└─ Prompt: "用户说'[用户描述]'，这听起来是解决方案。
   请帮我设计5个问题来挖掘真实需求，用5个Why方法"

第二步：AI生成深挖问题
示例：用户说"我需要搜索框"
├─ Why1: 为什么需要搜索？→ 因为找信息太慢
├─ Why2: 为什么找信息慢？→ 数据太多，要翻很多页
├─ Why3: 为什么要翻很多页？→ 没有分类，也不能筛选
├─ Why4: 为什么没有分类？→ 数据来源杂乱
└─ Why5: 为什么数据杂乱？→ 缺乏数据治理规范

第三步：真实需求浮现
└─ 真实需求：提高信息检索效率（搜索只是一种解决方案）
└─ 可能的更好方案：分类+筛选+搜索的组合，或者智能推荐
```

**AI输出示例：**
```
访谈提纲生成：

1. 能简单介绍一下您的日常工作流程吗？
2. 最近工作中有什么让您印象深刻的事情？

6. [Who] 除了您，还有谁会用到这个功能？使用频率？
7. [When] 您通常在什么时候会需要这个功能？
8. [Where] 您主要在什么场景下使用？办公室？外出？移动端？
9. [How] 您现在是怎么处理这个问题的？有什么不方便？

【场景化问题-STAR】

【优先级确认】

### 🤔 常见问题9：多方需求冲突，不知道如何协调？

**AI辅助冲突分析和调解：**
```
Prompt: "我遇到需求冲突，请帮我分析并给出调解方案：
冲突方A（产品经理）：要求[需求A]，理由[...]
冲突方B（运营团队）：要求[需求B]，理由[...]
冲突点：[两个需求相互矛盾]

请分析：
1. 冲突的根本原因（表面vs深层）
2. 双方真实诉求
3. 3个可能的解决方案（技术折中、分期实现、重新定义）
4. 每个方案的利弊
5. 给我调解会议的主持话术"

AI输出：
【冲突根因分析】
表面：功能互斥
深层：双方对优先级的理解不同
  - 产品经理关注用户体验
  - 运营团队关注业务转化
  
【解决方案】
方案1：技术上同时支持
  - 实现：用配置项，不同用户看到不同功能
  - 优点：双方都满意
  - 缺点：开发量大（+40%），维护复杂
  - 时间：+2周

方案2：AB测试决策
  - 实现：先上A方案，收集数据，再决定是否切换
  - 优点：用数据说话，避免主观争执
  - 缺点：决策周期长
  - 时间：正常，但结论需要1个月

方案3：分期实现
  - 一期：A的需求（对用户体验影响大）
  - 二期：B的需求（运营工具）
  - 优点：都能实现，缓解矛盾
  - 缺点：B方需要等待
  - 时间：一期正常，二期+1个月

【推荐】方案2（AB测试）
理由：避免主观判断，用数据驱动决策，长期看最优

【调解话术】
"感谢两位的输入，两个需求都很重要。我们分析后发现，冲突的根源是[...]

我建议我们用AB测试的方式：
- 先实现产品经理的方案，面向50%用户
- 同时收集运营团队关注的转化数据
- 4周后根据数据决定：是全量推广A，还是改成B，还是共存

这样的好处是：
1. 让数据而不是意见来决策
2. 两个方案都有机会验证
3. 我们学到真实的用户反馈

两位觉得这个方案如何？"
```
